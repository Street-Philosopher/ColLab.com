MyCashier

MyCashier is a program that is designed to help small businesses keep track of their sales and inventory. It is a simple program that began as a proof of concept and has grown into a full fledged program. It is designed to be easy to use and easy to understand. It is also designed to be as customizable and modifiable as per user needs. MyCashier is 100% free and open source. It is designed to be used by anyone and everyone. It is available on windows, and linux with python source code available for all other platforms.
